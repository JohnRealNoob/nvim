### Requirements

- [Git](https://git-scm.com/)
- [Neovim](https://github.com/neovim/neovim)

### Enter Your .config directory

```sh
cd .config
```

### Clone Repository

```sh
git clone https://github.com/JohnRealNoob/nvim.git
```

### Enter Neovim

```sh
nvim .
```
